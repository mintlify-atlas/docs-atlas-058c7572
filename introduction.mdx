## Welcome to Alefoods

Alefoods is a modern digital showroom built to showcase premium Peruvian superfoods and connect exporters with global buyers. The platform combines a beautiful product catalog, inspiring recipe library, and streamlined lead generation system to help you expand your reach in international markets.

<Info>
  Built with Astro 5, Tailwind CSS 4, and powered by Content Island CMS for seamless content management.
</Info>

## What is Alefoods?

Alefoods serves as your digital storefront for Peruvian exports, featuring:

- **Ready-to-eat quinoa** (red and white varieties)
- **Specialty coffee** from Oxapampa
- **Natural Peruvian superfoods**

The platform is designed to attract international buyers, showcase product quality with rich media, and capture qualified leads through an integrated contact system.

## Architecture highlights

The application follows a pod-based architecture for clean separation of concerns:

```typescript src/lib/client.ts:1
import { createClient } from "@content-island/api-client";
import { CONTENT_ISLAND_SECRET_TOKEN } from "astro:env/server";

const client = createClient({
    accessToken: CONTENT_ISLAND_SECRET_TOKEN,
})

export default client;
```

- **Pods**: Self-contained modules combining data fetching, models, and presentation
- **Server actions**: Form handling with built-in validation and error handling
- **Type-safe APIs**: TypeScript interfaces for products and recipes
